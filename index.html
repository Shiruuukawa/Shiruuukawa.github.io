About The BSCS

About this project :

The BSCS is a simple web platform built for BSCS 2 - 1 students. This website helps you manage your school tasks, check your weekly schedule, and set reminders for important deadlines.

The Task page lets you add and manage pending activities using your nickname. No passwords are required. This makes it easy for everyone in the class to participate and stay organized.
The Schedule page displays the official Monday to Saturday class timetable. It helps you see your subjects and time slots at a glance so you stay prepared every day.

The Reminder page allows you to set alerts for upcoming tasks or events. You can type a reminder, choose a date and time, and receive a notification with a sound when it is due.

This website was developed to improve productivity, organization, and communication within the class. It serves as a shared digital space where students can stay updated and manage their academic responsibilities efficiently.
